# WEDA-16 — Frontend Artifact Bundle Drop Point

Place or link the FE artifact bundle here for CTO review (WEDA-15).

Expected contents
- `bundle/` (source code or zipped patch set)
- `RUN.md` (install/run instructions)
- `QA-NOTES.md` (known issues, test notes)
- `reports/lighthouse/*.json` (Lighthouse/Web Vitals snapshots)
- `reports/a11y/*.json` (axe or similar a11y scan results)

How to use
1. Drop the files in this folder (or commit a branch link in `RUN.md`).
2. Ping WEDA-15 with a link to this folder and a short summary.
3. CTO will run the review against this exact snapshot.
